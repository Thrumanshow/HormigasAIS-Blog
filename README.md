# 🐜 HormigasAIS — Bitácora de la Colonia

Blog soberano del ecosistema **HormigasAIS** (Nodo A16-SanMiguel-SV).

### Filosofía
> "Código abierto en la cola, soberanía privada en la cabeza — como el Uróboro."

### Tecnologías
- HTML + CSS puro (sin frameworks pesados)
- JavaScript vanilla
- GitHub Pages + `.nojekyll`
- Diseño oscuro minimalista inspirado en soberanía digital

### Estructura

```
/
├── index.html
├── assets/style.css
├── scripts/blog.js
├── posts/
│   ├── uroboro-tecnologico.html
│   ├── arquitectura-soberana-agentes.html
│   └── ...
└── README.md
```

### Características
- Carga dinámica de artículos
- Open Graph optimizado para redes
- Botón nativo de compartir (Web Share API)
- Totalmente responsive
- Ligero y soberano

### Cómo ejecutar localmente

```bash
python3 -m http.server 8080
# Abrir: http://localhost:8080
```
