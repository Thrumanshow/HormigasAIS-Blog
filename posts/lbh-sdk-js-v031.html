LBH-SDK-JS v0.3.1: Modularidad, Tests de Conformidad y Preparación ante cambios de npm

19 de julio de 2026 | Nodo A16-SanMiguel-SV

1. ¿Qué cambia en esta versión?

La versión 0.3.1 reorganiza el SDK en una estructura modular clara, añade suite completa de tests de conformidad con LBH_SPEC_v2.0 y prepara el paquete para los próximos cambios de seguridad en npm v12.

Mejoras principales

  • Estructura modular (src/constants.js, src/exceptions.js, src/index.js)
  • 25 tests automáticos de conformidad
  • Manejo seguro de HMAC-SHA256 (timing-safe)
  • Mejor documentación y preparación para Trusted Publishing

2. Instalación

npm install lbh-sdk-hormigasais

3. Ejemplo: TOTP + LBH

Integración de autenticación de dos factores con el protocolo LBH:

// Generar TOTP + sellar en paquete LBH
const { frame, totpCode } = createSecureLBHFrame('2fa_verify');

// Verificar
const result = verifyLBHWithTotp(frame, userToken);
